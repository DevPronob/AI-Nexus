# AI Nexus Platform

A premium, full-stack AI-powered consultancy platform built with Next.js 14, Express.js, and MongoDB.

## 🚀 Features
- **AI-Powered Expert Matching**: Find the perfect consultant using our proprietary algorithm.
- **Premium UI/UX**: High-end glassmorphism design with Framer Motion animations.
- **Role-Based Dashboards**: Secure portals for both users and administrators.
- **Real-time Analytics**: Performance tracking and revenue overview charts.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 (App Router), Tailwind CSS v4, Lucide Icons, Recharts, Framer Motion.
- **Backend**: Node.js, Express.js, TypeScript, Mongoose.
- **Database**: MongoDB Atlas.
- **State Management**: Zustand & React Query.

## 🏁 Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account (or local MongoDB)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/DevPronob/AI-Nexus.git
   ```
2. Install dependencies:
   ```bash
   # Root
   npm install
   # Frontend
   cd frontend && npm install
   # Backend
   cd ../backend && npm install
   ```

### Running the App
1. Start the backend:
   ```bash
   cd backend && npm run dev
   ```
2. Start the frontend:
   ```bash
   cd frontend && npm run dev
   ```
